Para esta práctica tenemos que hacer un caso de pruebas de Junit para un programa java
que recibe tres numeros y los ordena de mayor a menor , para ello pasa por dos bucles if en el que se ordenan
progresivamente if(x < y) y a su vez (z < y) y es el valor mas grande.
Esto se repite tres veces en los bucles que aparecen en el diagrama y las posibles rutas de ejecucion están cubiertas 
en la tabla de pruebas añadidos ambos en la carpeta DATOS_ADICIONALES tambien disponible en este repositorio.
Nos aseguramos de tener una cobertura del 100%.
